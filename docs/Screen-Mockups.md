# Screen Mockups (Box Drawing)

These mockups use box-drawing characters (`┌`, `─`, `┐`, `└`, `┘`, `│`) for borders. In-game pieces render as background-colored spaces via ncurses color pairs; `█` is used here as a visual placeholder. Empty cells use `·` as a placeholder; in-game they are blank spaces. Final layout may adjust based on terminal size.
